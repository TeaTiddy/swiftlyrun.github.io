Update apt
```bash
apt update
```
Install available package upgrades
```bash
apt upgrade
```
Install package `curl`
```bash
apt install curl
```
Uninstall package `curl`, leaving config files
```bash
apt remove curl
```
Uninstall package `curl`, removing config files
```bash
apt purge curl
```
